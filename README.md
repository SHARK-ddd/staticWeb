# 烟花秀网页

一个可以在手机上运行并能在微信中直接打开的烟花秀网页。

## 功能特点

- 📱 **响应式设计**：适配各种手机屏幕尺寸
- 💫 **流畅的烟花动画**：基于Canvas实现的绚丽烟花效果
- 📲 **微信友好**：支持直接在微信中打开访问
- 🎮 **交互控制**：可以手动开始和停止烟花表演
- 🎨 **美观界面**：简洁现代的设计风格

## 快速开始

### 方法一：直接运行启动脚本

1. 在当前目录下运行 `start_server.py`：
   ```bash
   python start_server.py
   ```

2. 根据终端提示访问网页：
   - 本地访问：`http://localhost:8000`
   - 手机访问：`http://[你的电脑IP]:8000`（需要在同一局域网内）

### 方法二：手动启动HTTP服务器

1. 在当前目录下启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```

2. 访问 `http://localhost:8000` 或 `http://[你的电脑IP]:8000`

## 在微信中使用

1. 确保手机和电脑在同一局域网内
2. 查看电脑的IP地址（可以通过 `ipconfig` 命令查看）
3. 在微信中打开链接：`http://[电脑IP]:8000`
4. 即可在微信中直接观看烟花秀

## 如何获取电脑IP地址

### Windows系统
1. 按下 `Win + R` 键，输入 `cmd` 并回车
2. 在命令提示符中输入 `ipconfig` 并回车
3. 查找 `IPv4地址` 项，后面的数值就是你的电脑IP地址

### macOS系统
1. 打开终端
2. 输入 `ifconfig` 并回车
3. 查找 `inet` 项，后面的数值就是你的电脑IP地址

## 自定义配置

你可以修改 `index.html` 文件中的以下参数来自定义烟花效果：

- 烟花颜色：修改 `getRandomColor()` 方法中的颜色数组
- 烟花数量：调整 `animate()` 函数中的生成频率
- 粒子数量：修改 `createParticles()` 方法中的粒子数量

## 注意事项

1. 确保电脑和手机在同一局域网内，否则手机无法访问
2. 部分公共网络可能会限制设备间的访问
3. 如果遇到访问问题，可以尝试关闭电脑的防火墙
4. 建议使用现代浏览器访问，以获得最佳体验
